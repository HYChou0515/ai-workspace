# 開始使用 / Getting started

歡迎使用這個工作平台。這裡把你的多個 App 與共享的知識庫(Knowledge Base）放在一起。

Welcome. This platform brings your apps and a shared Knowledge Base together in
one place.

## 平台是什麼 / What this platform is

- **App(應用）**:每個 App 是一個聚焦的工作區。從首頁的卡片點進去就能開始。
  Each app is a focused workspace — open one from a card on the home screen.
- **知識庫 / Knowledge Base**:共享的文件、wiki 與聊天都在這裡。文件上傳後會被索引,
  之後可以用 AI 帶引用地問答。
  Shared documents, wikis, and chat live in the Knowledge Base. Uploaded
  documents are indexed so the AI can answer questions about them with citations.

## 怎麼開始 / How to start

1. 從首頁(Launcher）挑一個 App,或進入知識庫。
   Pick an app from the launcher, or open the Knowledge Base.
2. 在知識庫建立一個 collection、上傳文件,文件就緒後即可被搜尋。
   Create a collection in the Knowledge Base and upload documents; once they are
   ready they become searchable.
3. 在任何聊天視窗用自然語言提問。回答會附上 `[n]` 引用,點擊可跳到來源。
   Ask questions in natural language in any chat. Answers cite their sources as
   `[n]`, which you can click to open.

## 需要協助時 / When you need help

- 點頂部導覽列的 **Help** 隨時回到這個說明頁。
  Click **Help** in the top navigation bar to return to this page anytime.
- 在這個說明頁直接問 AI「怎麼……」,它會依使用說明與更新紀錄回答。
  Ask the AI here "how do I…" and it will answer from this guide and the
  release notes.
